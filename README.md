# Scan_resume - 智能简历解析与扫描系统
一款基于前后端分离架构的**PDF简历智能解析工具**，通过PDF文本提取 + AI自然语言处理技术，自动抓取简历中的核心信息，实现简历快速筛选、结构化数据提取，大幅提升招聘/简历整理效率。

## ✨ 核心特性
- **PDF简历解析**：精准提取PDF格式简历的纯文本内容
- **AI智能提取**：自动识别并结构化输出姓名、电话、邮箱、工作经历、技能、学历等核心信息
- **前后端分离**：可视化操作界面，零门槛使用
- **轻量高效**：部署简单，响应快速，支持单份/批量简历处理
- **结构化输出**：解析结果标准化，方便二次使用与数据导出

## 🛠️ 技术栈
### 后端
- 语言：`Python 3.8+`
- 核心模块：PDF文本解析、AI信息提取、接口服务
- 依赖管理：`requirements.txt`

### 前端
- 语言：`TypeScript`
- 构建工具：`Vite`

## 📂 项目结构
```
Scan_resume/
├── backend/                # 后端服务模块
│   ├── ai_extractor.py     # AI核心：简历信息智能提取
│   ├── pdf_parser.py       # PDF简历文本解析
│   ├── main.py             # 项目入口 & 接口服务
│   └── requirements.txt    # 后端依赖清单
└── frontend/               # 前端交互界面
    ├── src/                # 前端源码
    ├── index.html          # 入口页面
    ├── package.json        # 依赖&脚本配置
    ├── tsconfig.json       # TS编译配置
    └── vite.config.ts      # Vite构建配置
```

## 🚀 快速开始
### 环境要求
1. Python 3.8 及以上版本
2. Node.js 16.0 及以上版本

---

### 1. 启动后端服务
```bash
# 进入后端目录
cd backend

# 安装依赖
pip install -r requirements.txt

# 启动服务
python main.py
```

### 2. 启动前端服务
```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 启动开发环境
npm run dev
```

### 3. 访问使用
前端默认地址：`http://localhost:3000`（终端会显示具体端口）
上传PDF简历 → 自动解析 → 查看结构化结果

## 📝 使用说明
1. 确保前后端服务均正常启动
2. 在前端界面点击**上传简历**，选择本地PDF格式简历
3. 系统自动完成解析与AI信息提取
4. 页面实时展示提取后的结构化信息

## ❓ 常见问题
1. **PDF解析失败？**
   仅支持标准文本型PDF，暂不支持图片扫描版简历
2. **前后端无法通信？**
   确认后端服务正常运行，检查接口地址配置
